# カメラ映像と温度グラフのリアルタイム表示

ここでは、3dpmon (v2.1.004) で提供されるリアルタイム監視機能について説明します。カメラ映像と温度グラフを組み合わせることで、複数プリンタの印刷状況を直感的に把握できます。

## 主な機能

### カメラストリーム
- プリンタに搭載されたカメラ映像をパネル上でストリーム再生します。
- **プリンタごとのカメラストリーム**: 各プリンタのカメラ映像は独立したパネルに表示されます。
- **Per-host カメラ ON/OFF 制御** (`hostCameraToggle`): プリンタごとにカメラストリームの有効・無効を切り替えられます。帯域やリソースの節約に有効です。

### 温度グラフ
- ノズル温度とベッド温度を折れ線グラフで表示し、目標温度と現在温度を比較できます。
- **各プリンタ独立の温度グラフ**: 温度データは `_hostChartData` Map でプリンタごとに分離管理されます。複数プリンタの温度推移が混在することはありません。
- 温度グラフは履歴を保持し、経過時間ごとの推移を確認可能です。

### 更新サイクル
- `dashboard_aggregator.js` が **500ms 間隔** で全ホストのデータを走査し、変更があったフィールドのみ DOM を更新します（変更キュー `_dirtyKeys` による差分更新）。

## パネルシステム
リアルタイム表示はすべて GridStack パネルとして構成されます。カメラパネル・温度グラフパネルはパネルメニューから追加でき、自由にレイアウトを変更できます。

## 操作のヒント
- パネル内のカメラ映像はクリックで拡大表示できます。
- 温度グラフはマウスオーバーで現在値をツールチップ表示します。
- **Per-host グラフリセット**: 温度履歴をリセットしたい場合は、対象プリンタのパネルから **グラフリセット** を実行してください。プリンタごとに個別にリセットできます。
