# ブラウザ / Electron から Creality シリーズを監視する

このドキュメントでは、3dpmon (v2.1.004) を利用して Creality シリーズの 3D プリンタをブラウザまたは Electron アプリ上で監視する方法を説明します。

## 対応機種
- Creality K1 / K1C / K1A / K1 Max 系列

## 実行環境
- **ブラウザ**: `3dp_monitor.html` を直接開いて使用します。
- **Electron アプリ**: `electron/main.js` から起動します。Electron 環境では GridStack によるパネルの自由配置が利用でき、ウィンドウサイズの保存やネイティブ通知にも対応します。

## 基本的な接続フロー
1. アプリを起動します（ブラウザで `3dp_monitor.html` を開くか、Electron アプリを起動）。
2. **接続管理モーダル** を開き、プリンタの IP アドレスとポート（デフォルト `9999`）を入力して接続先を追加します。
3. 接続に成功するとパネルにホスト名と状態が表示され、カメラ映像や各種情報の取得が始まります。

## マルチプリンタ管理
v2.1.004 では複数のプリンタを同時に監視できます。

- **接続先の追加・削除・再接続**: 接続管理モーダルから複数のプリンタを登録し、個別に接続・切断・再接続が行えます。
- **接続状態インジケータ**: 各接続先の状態が色分け表示されます（接続中 / 切断 / 再接続中）。
- **Per-host 接続カラー設定**: プリンタごとに識別用の色を割り当てることで、パネル上でどのプリンタの情報かを一目で判別できます。
- **Per-host データ分離**: 各プリンタのデータは `monitorData.machines[hostname]` で独立管理され、データの混在はありません。

## パネルシステム
監視情報は GridStack ベースのパネルとして表示されます。パネルは自由にドラッグ・リサイズでき、レイアウトは自動保存されます。パネルの追加はパネルメニュー (`dashboard_panel_menu.js`) から行います。

## 接続がうまくいかない場合
- IP アドレスが正しいか、プリンタと同じネットワークに接続されているか確認してください。
- WebSocket のポート番号は通常 `9999` で自動設定されます。変更している場合は接続管理モーダルで調整してください。
- 接続が切れた場合は自動再接続が試みられます。手動で再接続する場合は接続管理モーダルから操作してください。

ブラウザまたは Electron アプリでの監視により、離れた場所からでも複数プリンタの状態を同時に確認できます。
