# 3dpmon 操作ガイド

## 動作仕様
- Creality K1/K1C/K1A/K1 Max シリーズに対応した監視ダッシュボード
- WebSocket (標準ポート 9999) を利用してプリンタと通信
- **複数台のプリンタを同時に監視可能** (per-host データ分離)
- カメラ映像、温度グラフ、印刷状態をリアルタイム表示
- 遠隔操作、ファイル管理、通知機能を搭載
- GridStack によるパネルの自由配置 (ドラッグ&リサイズ対応)
- Electron アプリまたはブラウザから利用可能
- 設定や履歴はブラウザの IndexedDB / localStorage に保存されます
- 準備時間、1層目確認時間、一時停止時間、使用フィラメント情報も
  印刷履歴に記録されます
- これらの時間はページを再読み込みしても保持されます

## 起動方法

### 方法 1: Electron アプリ (推奨)
1. リポジトリをダウンロードし、任意のフォルダに配置します
2. 依存パッケージをインストールします:
   ```
   npm install
   ```
3. Electron アプリを起動します:
   ```
   npm run electron
   ```
4. アプリウィンドウが開き、ダッシュボードが表示されます

### 方法 2: ブラウザ (HTTP サーバー)
1. 静的 HTTP サーバーを起動します。例:
   ```
   python -m http.server 8000
   ```
   Windows であれば `start.bat` をダブルクリックするだけでもサーバーを起動できます。
2. ブラウザで `http://localhost:8000/3dp_monitor.html` を開きます

## 接続管理 (マルチプリンタ対応)

### 接続管理モーダル
複数台のプリンタを同時に監視するには、接続管理モーダルを使用します。

1. タイトルバーの接続管理ボタンをクリックしてモーダルを開きます
2. プリンタの IP アドレスまたはホスト名を入力し、**追加** をクリック
3. 追加されたプリンタごとに **接続** ボタンで接続を開始します
4. 複数のプリンタを同時に接続し、それぞれ独立して監視できます

対応機種 (K1/K2 シリーズ) ではポートは自動設定されるため、IP のみで接続できます。
カメラ映像は現在 `8080` ポートのみ対応しています。

### per-host データ分離
各プリンタは完全に独立したデータを持ちます:
- 温度データ、印刷状態、ファイルリストはプリンタごとに分離
- カメラの ON/OFF はプリンタごとに個別設定
- フィラメント装着状態 (`hostSpoolMap`) はプリンタごとに管理
- 3D プレビューの回転状態もプリンタごとに保持

## GridStack パネルシステム

### パネルの追加
- パネル追加メニューから必要なパネルを選んで追加します
- 各パネルは特定のプリンタ (ホスト) に紐付きます
- 同じ種類のパネルを複数追加し、異なるプリンタを並べて比較できます

### パネルの移動・リサイズ
- パネルのタイトルバーをドラッグして自由に配置できます
- パネルの端をドラッグしてリサイズできます
- レイアウトは自動的に保存され、次回起動時に復元されます

### パネル一覧
- **監視パネル** -- カメラ映像と印刷操作
- **温度グラフ** -- ノズル・ベッド温度の推移
- **ヘッドプレビュー** -- ヘッド位置のリアルタイム表示
- **フィラメントプレビュー** -- スプール残量の 3D 表示
- **操作パネル** -- ファン・LED・温度の調整
- **機器情報** -- ハードウェア詳細
- **印刷履歴** -- ジョブ履歴とファイル管理
- **状態パネル** -- 進捗と残り時間

## 音声のアンロックとカスタマイズ
ブラウザの仕様により、ユーザーが画面をクリックするまで音声を再生
できません。起動画面のどこでもクリックするとアンロックされ、右下
に音楽と音声のアイコンが表示されます。アイコンが斜線付きの場合は
無効、丸印なら有効で、通知時に音声や効果音が再生されます。

## 通知設定
通知設定は **接続管理モーダルのサブモーダル** に移動しました。
プリンタごとに以下の設定が可能です:
- 効果音・音声通知の有効/無効
- 異常温度の警告閾値
- TTS 音声・速度の per-host 設定 (各プリンタに異なる音声を割り当て可能)
- カテゴリ別通知 (印刷イベント / カメラ / 温度アラート)

## 画面構成
- **タイトルバー** -- プリンタ名や印刷状態を表示し、接続管理ボタンがあります。音声が無効な場合はミュート表示が出ます。
- **監視パネル** -- カメラ映像、ヘッド位置プレビュー、印刷状態テーブル、停止・一時停止ボタンなどをまとめています。ノズルやベッド温度の調整、各ファンや LED の切り替えも行えます。
- **温度グラフ** -- ノズルとベッドの温度推移を折れ線グラフで表示します。
- **機器情報パネル** -- 加速度制限やモデル情報、使用統計を確認できます。
- **印刷履歴パネル** -- 印刷履歴一覧とファイル一覧タブがあり、G-code アップロードも可能です。
- **設定パネル** -- ストレージ設定やコマンドパレットなどをまとめたエリアです。

## フィラメントスプール管理
- フィラメント管理モーダルの **追加** ボタンからスプールを登録します。ダイアログでは
  スプール名、総長さ、現在の残り長さを入力します。
- 旧データは起動時に自動で新形式へ変換されます。
- 複数プリンタ環境では、各プリンタに異なるスプールを装着できます (`hostSpoolMap`)。
- 印刷完了後は残量が自動更新され、3D プレビューで確認できます。
- 印刷中も装置が報告する使用量を基に残量を減算し、数分おきに保存します。
- 登録ダイアログでは以下の項目を入力できます:
  - スプール名・サブ名
  - メーカー名 (任意)
  - 材料名と色名
  - 巻き長さと重量 (m/g は自動変換)
  - HEX カラーコード

  例: フィラメント登録時のオーバーレイ

  ```
  +-----------------------------+
  | Name        [Generic PLA ]  |
  | Sub Name    [Orange ]       |
  | Manufacturer[ACME]          |
  | Material    [PLA ] (+)      |
  | Length      [1.0] kg = 330m |
  | Color (#)   [#f97316]       |
  +-----------------------------+
  ```

  新しいメーカー名やカスタム素材を追加したい場合は、入力欄右側の [+] ボタンから
  直接登録できます。次回以降はプルダウンから選択可能です。
