# 遠隔操作とファイル管理、通知設定

このセクションでは、3dpmon (v2.1.004) の遠隔操作機能、ファイル管理機能、通知機能の詳細を説明します。すべての操作はマルチプリンタ環境に対応しています。

## 遠隔操作
- **操作パネル** からノズル・ベッド温度の変更、ファンや LED の切り替えが可能です。
- G-code 実行中でも一時停止や停止、リセットコマンドを送信できます。
- K1 Max、K1、K1C、K1A では一時停止中に原点復帰と待機位置移動を行う **一時停止時 原点復帰** ボタンが利用できます。
- 一時停止時にヘッドを手動で動かしたい場合は **XYロック解除** ボタンでステッピングモーターのロックを解除できます。
- よく使うコマンドは **コマンドパレット** にまとめられており、ワンクリックで実行できます。

### Per-host コマンドルーティング
v2.1.004 では、コマンドは各パネルが紐付いているホストに対して送信されます。操作パネルから送ったコマンドは、そのパネルが表示しているプリンタにのみ届きます。異なるプリンタへの誤送信は発生しません。

## ファイル管理
- **ファイル一覧** パネルから G-code ファイルのアップロードと削除が行えます。
- ドラッグ&ドロップにも対応しており、誤って表示された場合はオーバーレイ右上の × ボタンでキャンセル可能です。
- アップロード済みのファイルを選択し、**印刷開始** ボタンでジョブを実行します。
- **Per-host ファイルリスト**: ファイル一覧は `_fileListMap` により各プリンタごとに分離管理されます。プリンタ A のファイルリストとプリンタ B のファイルリストは独立しています。

### 印刷履歴
- 印刷履歴パネルから過去に印刷したファイルを再実行することも可能です。
- **Per-host 印刷履歴追跡**: 印刷履歴はプリンタごとに記録されます。どのプリンタでいつ何を印刷したかを個別に確認できます。

## 通知設定
通知設定は **接続管理モーダルのサブモーダル** から行います（v2.1.004 で設定パネルから移動）。

- テーブル形式でカテゴリ別に折り畳み表示されます（印刷イベント / カメラ / 温度アラート）。
- 印刷完了やフィラメント残量低下をブラウザ通知または Webhook 経由で受信できます。
- 残量警告の閾値や Webhook URL は通知設定サブモーダルから編集可能です。

### Per-host TTS 設定
- **機器ごとに音声を変更可能**: TTS（音声読み上げ）の音声・速度はプリンタごとに個別設定できます（`_hostTts` Map で管理）。
- 例えば、プリンタ A は女性音声、プリンタ B は男性音声のように使い分けることで、音声だけでどのプリンタの通知かを判別できます。
- `notify()` は `payload.hostname` から対応する per-host TTS 設定を自動適用します。

## マルチプリンタ管理
接続管理モーダルから複数プリンタの接続を一元管理します。

- 接続先の追加・削除・再接続がモーダル内で完結します。
- 各プリンタの接続状態（接続中 / 切断 / 再接続中）がリアルタイムで表示されます。
- パネルは GridStack により自由配置でき、複数プリンタの情報を並べて表示・比較できます。
