# フィラメント管理 UI/UX 再設計計画

## 現状の問題分析

### ユーザーからのクレーム要約

> フィラメント管理やコスト計算ができるのが売りなのに、UIUXがおざなりなせいで
> 管理機能が貧弱という評価。ユーザー評価の星スコアを大きく下げられている。
> パネル群はシングル監視時代のUIを解決してきたのに、フィラメント交換・管理画面だけ改善されていない。

### 根本原因

1. **フィラメント交換フローが3つに分散し、それぞれの違いが不明**
   - 「交換」ダイアログ → 登録済みスプールから選択
   - 「過去取り外したスプールから選択」→ Tab 2 を開くだけ (交換と実質同じ)
   - 「新品を開封」→ プリセット選択ダイアログ → 交換ダイアログに戻る (2段階)

2. **モーダル in モーダルの操作感が悪い**
   - 交換ダイアログ → 新品開封ダイアログ → 元に戻る → 装着
   - ユーザーは「交換したい」だけなのに3ステップ必要

3. **情報の密度が低く画面に収まらない**
   - テーブルが縦長で一覧性が悪い
   - プレビューが120x120で小さすぎて情報を伝えられない
   - アイコンやバッジが活用されていない

4. **プレビュー連動の不完全**
   - 交換後にフィラメントパネルに反映されないケースがある
   - Tab 2 の「装着」がプレビューに反映されない

5. **お気に入りプリセットのクイック選択が消失**
   - 以前あったお気に入りプリセットから1クリックで交換する機能がない
   - 新品開封のたびに全プリセットから探す必要がある

## 改善方針

### コンセプト: 「1画面で交換完結」

交換ダイアログを3つのタブに統合し、1つのダイアログ内で全操作を完結させる。

### 新しい交換ダイアログ構造

```
┌─ フィラメント交換 — K1Max-4A1B ──────────────────────────────┐
│                                                              │
│ ■ 現在装着中: #001 Emerald PLA ■ 45% (151.2m)               │
│   [取り外す]                                                 │
│                                                              │
│ ┌──────────┬──────────┬──────────┐                           │
│ │ 保管中    │ 新品開封  │ お気に入り │                          │
│ └──────────┴──────────┴──────────┘                           │
│                                                              │
│ [検索: ブランド▼ 材質▼ 色▼ 名称_____ 🔍]                    │
│                                                              │
│ ┌─────────────────────────────────────────────────────────┐  │
│ │ ■ #003 Navy PETG   85% ██████████░░  PRINSFIL          │  │
│ │ ■ #005 Red PLA+    60% ██████░░░░░  CC3D               │  │
│ │ ■ #007 Black ABS   30% ███░░░░░░░░  PRINSFIL  [他機器] │  │
│ └─────────────────────────────────────────────────────────┘  │
│                                                              │
│                    [キャンセル]  [この フィラメントに交換]     │
└──────────────────────────────────────────────────────────────┘
```

### タブの役割

| タブ | 内容 | 対象 |
|------|------|------|
| 保管中 | removedAt あり + 残量 > 0 のスプール | 既に使った、取り外し済みのスプール |
| 新品開封 | プリセット一覧 (開封即登録) | まだ使ったことのないスプール |
| お気に入り | isFavorite=true のスプール + お気に入りプリセット | よく使うものへのショートカット |

### 各行の表示

カード風の行で、アイコン+バッジを活用:

```
┌─────────────────────────────────────────────────────────────┐
│ ■ #003 Navy                                                 │
│   PRINSFIL PETG  85%  ██████████░░  268.0m (427g, ¥2,550) │
└─────────────────────────────────────────────────────────────┘
```

- 色スウォッチ (■) + シリアル番号 + 名称
- ブランド + 素材 + 残量バー + 人間可読残量 (formatFilamentAmount)
- 他機器装着中はグレーアウト + バッジ

### 実装フェーズ

#### Phase 1: 即時修正 (データフロー)
- 交換後のプレビュー反映漏れを修正
- 「過去取り外したスプールから選択」を交換ダイアログ内のタブ切替に変更

#### Phase 2: ダイアログ統合
- 3つのフロー (保管中/新品/お気に入り) を1つのダイアログに統合
- カード風行表示に変更

#### Phase 3: ビジュアル強化
- 残量バーのカラフル化
- アイコン活用 (装着状態、素材種類)
- レスポンシブ対応
